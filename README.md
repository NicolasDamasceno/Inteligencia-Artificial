# Estudos de Inteligência Artificial e Deep Learning 🤖

Este repositório contém o desenvolvimento de modelos de IA e Redes Neurais originalmente criados no Google Colab e migrados para um ambiente de desenvolvimento local (**VS Code**). O foco principal é a aplicação prática de Deep Learning e o pré-processamento de dados para modelos preditivos.

## 🚀 Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Deep Learning:** Keras & TensorFlow
- **Processamento de Dados:** Pandas & Sklearn (StandardScaler)
- **Visualização:** Matplotlib
- **Ambiente:** VS Code & Linux Mint

## 📚 Tópicos Abordados
- **Redes Neurais Sequenciais:** Construção de modelos camada por camada usando `Sequential`.
- **Otimização de Treinamento:** Uso de `EarlyStopping` para evitar overfitting e `Dropout` para regularização.
- **Pré-processamento:** Normalização de dados com `StandardScaler` e divisão de bases com `train_test_split`.
- **Avaliação de Modelos:** Análise de performance via `accuracy_score` e Matriz de Confusão.

## 📂 Estrutura do Repositório
- `/modelos`: Scripts Python (.py) para execução local.
- `/data`: Datasets utilizados para o treinamento dos modelos.
- `requirements.txt`: Lista de dependências necessárias para rodar o projeto.
